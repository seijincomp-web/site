# Solo Leveling - Minecraft Bedrock Mod

Mod inspirado na série "Solo Leveling" com sistema único de sombras aliadas.

## 🎮 Características Principais

### Sistema de Sombras Aliadas
- **Tela de Invocação**: Item especial no inventário que abre uma interface
- **Mobs Derrotados**: Apenas mobs que você eliminou podem ser invocados
- **Sombras Escuras**: Mobs invocados ficam com aparência escura (shadow)
- **Combate Aliado**: As sombras atacam automaticamente seus inimigos

## 📦 Estrutura do Projeto

```
site/

├── resource_packs/
│   └── solo_leveling/        # Texturas, sons e modelos
├── scripts/
│   └── summoning_system.js   # Sistema de invocação
└── docs/
    └── README.md             # Documentação
```

## 🚀 Como Começar

1. Copie as pastas `behavior_packs/solo_leveling` e `resource_packs/solo_leveling` para seu mundo
2. Ative o mod nas configurações do mundo
3. Comece a derrotar mobs para desbloquear suas sombras!
